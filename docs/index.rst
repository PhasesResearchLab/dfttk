Documentation
=================================
DFTTK (Density Functional ToolKit) is a flexible tool for automating density functional theory (DFT) calculations with VASP and storing results in MongoDB. It leverages [Custodian](https://github.com/materialsproject/custodian) for job submission and error handling, while [PyMongo](https://github.com/mongodb/mongo-python-driver) facilitates data storage and retrieval in MongoDB.
